How it works

1) Create a profile

Stoägare: add mares for free. Hingstägare: later pay for visibility + leads.

2) Use as a catalog

Filter by discipline, semen type, region, index, 3-year test and height interval.

3) Build a shortlist

Switch to Match mode to shortlist around a selected mare (scoring formula later).

Built for two user types

Mare owners

Free profile + mare library

  • Add multiple mares
  • Catalog filtering
  • Match shortlist (MVP)
  • Save mare 3-year test details (not used yet)
Stallion owners

Paid visibility + leads (future)

  • Pay to appear in catalog & match
  • Featured placement
  • Lead inbox + analytics
  • Semen availability updates
